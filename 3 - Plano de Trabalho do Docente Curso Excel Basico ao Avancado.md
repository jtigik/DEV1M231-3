
# Plano de Trabalho do Docente: Excel Intensivo para Advogados - Do Básico ao Avançado

**Inspirado e Adaptado do PPP e Plano de Curso "Excel Intensivo: Do Básico ao Avançado" do SENAC-RS**

Este plano detalha atividades para os 12 encontros, redistribuindo o conteúdo original das sessões 1-9 (até Power Query) para manter 48 horas. Expansão inclui revisões, casos adicionais e aprofundamentos práticos, preservando hands-on (1h teoria, 2h prática, 1h Q&A). Metodologia: Interativa, com subgrupos de 4-5; quizzes via Forms.
